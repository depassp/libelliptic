    class Point

    This class implements a tri-dimensional point in Z/pZ.

    Coordinates

    Coordinates are stored as accessible variables #X, #Y, and #Z.

    Z/pZ

    Coordinates are integers modulo p. You should set the @@base variable to p.

    Attributes

    X[RW]

    The coordinates of the point, modulo @@base.

    Y[RW]

    The coordinates of the point, modulo @@base.

    Z[RW]

    base=(base) click to toggle source

    Sets the base of Z/pZ currently used for points.

    WARNING: in order to set up a projective space for elliptic curve, the base should be a prime number.

    Examples

    Basic usage:

    Point.base = 13
    => true
Point.new 0, 1, 5 == Point.new 0, -12, 18
    => true
Point.base = 7
    => true
Point.new 0, 1, 5 == Point.new 0, -12, 18
    => false
