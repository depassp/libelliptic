    class ElliPoint

    This class implements a projective tri-dimensional point of an elliptic curve in Z/pZ.

    Coordinates

    Coordinates are stored as accessible variables @X, @Y, and @Z.

    Z/pZ

    Coordinates are integers modulo p. You should set the @@base variable to p.

    Projective space

    In this projective space, points whith 0 as third coordinate @Z are infinity points, while the others have homogeneous coordinates :x and :y.

    Elliptic curve

    The elliptic curve is defined by variables @@a and @@b. In order to obtain a valid elliptic curve, 4*@@a^3 + 27*b^2 should not be 0.

    Elliptic calculations

    Within this elliptic curve, you can perform operations on points, because they form an abelian group.

def ** p
        return -p if self.isInfty?
        return -self if p.isInfty?
        if self == p
                l = divide (3 * self.x**2 + @@a), (2 * self.y), @@base
                k = l**2 - 2 * self.x
                return ElliPoint.new k, (l * (k - self.x) + self.y)
        end
        return ElliPoint.infty if self.x == p.x
        l = divide (p.y - self.y), (p.x - self.x), @@base
        k = l**2 - self.x - p.x
        return ElliPoint.new k, (l * (k - self.x) + self.y)
end
