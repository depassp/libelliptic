    module Ellithmetic

    This module implements specific arithmetics methods.

    Division ring

    Every operation is performed in Z/pZ, which is a division ring when p is prime. It contains integers from 0 to p-1.

    base parameter

    Consequently, each function requires a base argument as p.

    divide(n, q, base) click to toggle source

    Computes the division of a number by another in Z/pZ

    Attributes

    • n - The number to divide. It will be reduced modulo p.

    • q - The number to divide by. It will be reduced modulo p. WARNING: passing 0 will raise a ZeroDivisionError!

    • base - The base for Z/pZ. WARNING: if p is not prime, results will be incoherent!

    Examples

    Basic usage:

    divide 4, 2, 11
    => 2
divide 3, 2, 11
    => 7

    You can compute any number, it will be reduced in Z/pZ:

    divide -1, 4, 5
    => 1
divide -666, 4, 5
    => 1

    Computes the inverse of any number in Z/pZ

    Attributes

    • n - The number to compute. It will be reduced modulo p. WARNING: passing 0 modulo p will raise a ZeroDivisionError!

    • base - The base for Z/pZ. WARNING: if p is not prime, results will be incoherent!

    Examples

    Basic usage:

    inverse 1, 11
    => 1
inverse 3, 5
    => 2
inverse 4, 5
    => 4

    You can compute any number, it will be reduced in Z/pZ:

    inverse -666, 5
    => 4

def inverse n, base
        n_ = n % base
        raise ZeroDivisionError,
              "divided by 0 modulo " + base.to_s if
                        n_ == 0
        return 1 if n_ == 1
        i = 2
        i += 1 until (i * n) % base == 1
        return i % base
end
